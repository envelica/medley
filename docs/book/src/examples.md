# Examples

All examples are runnable with `cargo run --example <name>`:

- `expr_pull`: Pull-parse and evaluate a small arithmetic expression.
- `csv_pull`: Stream CSV rows with bounded memory.
- `parse_small`: Micro-benchmark for small inputs.
- `parse_stream`: Streaming benchmark over 1MB data.
- `ast_visitor`: Traverse and inspect an AST.
- `ast_transform`: Transform an AST in-place with `VisitorMut`.
- `w3c_ebnf`: Demonstrates W3C EBNF constructs.

See also: `docs/grammar-macro-spec.md` for macro details.
